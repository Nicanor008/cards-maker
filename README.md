## Cards Maker
Let's make the cards

## Tools
- ReactJS
- Bulma
- NodeJS - ExpressJS
- MongoDB - mlab

## Getting Started
- **yarn client** - run frontend on development
- **yarn server:dev** - run backend on development
- **yarn start:Dev** - run both frontend and backend on development
